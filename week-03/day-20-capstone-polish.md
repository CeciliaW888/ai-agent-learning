### Step 1: Write a Great README

```markdown
# 🤖 Multi-Agent Content Pipeline

A multi-agent system that researches, writes, and edits articles 
using specialized AI agents.

## How It Works

```
User → Researcher → Writer → Editor → (revision loop) → Final Article
```

Three AI agents collaborate to produce high-quality content:
- **Researcher** searches the web and gathers facts
- **Writer** crafts engaging articles from research
- **Editor** reviews quality and requests revisions

## Quick Start

```bash
# Clone and setup
git clone https://github.com/YOUR_USERNAME/content-pipeline.git
cd content-pipeline
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Add your API keys
cp .env.example .env
# Edit .env with your keys

# Run
python main.py "The future of AI in healthcare"
```

## Requirements

- Python 3.10+
- OpenAI API key
- Tavily API key (free tier: [tavily.com](https://tavily.com))

## Usage

```bash
# Basic usage
python main.py "Your topic here"

# With options
python main.py "AI agents" --requirements "Focus on practical applications"
python main.py "Remote work" --max-revisions 3 --verbose
```

### Step 4: Test Suite

```python
# tests/test_pipeline.py
"""
Quick tests to verify the pipeline works.
Run: python -m pytest tests/ -v
"""
import pytest
from pipeline import build_pipeline, PipelineState

def test_pipeline_builds():
    """Pipeline should compile without errors."""
    pipeline = build_pipeline()
    assert pipeline is not None

def test_pipeline_runs():
    """Pipeline should produce output for a simple topic."""
    pipeline = build_pipeline()
    
    result = pipeline.invoke({
        "topic": "Why Python is popular",
        "requirements": "Keep it brief",
        "research": "",
        "sources": [],
        "draft": "",
        "feedback": "",
        "approved": False,
        "revision_count": 0,
        "final_article": "",
        "metadata": {}
    })
    
    assert result["final_article"], "Should produce a final article"
    assert len(result["final_article"]) > 100, "Article should have substance"

def test_metadata_populated():
    """Pipeline should track metadata."""
    pipeline = build_pipeline()
    
    result = pipeline.invoke({
        "topic": "Testing AI agents",
        "requirements": "",
        "research": "",
        "sources": [],
        "draft": "",
        "feedback": "",
        "approved": False,
        "revision_count": 0,
        "final_article": "",
        "metadata": {}
    })
    
    meta = result.get("metadata", {})
    assert "word_count" in meta, "Should track word count"
    assert meta["word_count"] > 0, "Should have words"
```
