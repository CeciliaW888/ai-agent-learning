## About This Course

21 days. 21 hours. From AI user to AI agent builder.

This is a structured learning course that takes you from understanding what AI agents are, to building your first LangChain agent, to deploying a multi-agent content creation pipeline. One hour a day, video-heavy, hands-on projects, and designed to build in public.

**Duration:** 21 days (3 weeks)
**Daily commitment:** 1 hour
**Language:** English
**Level:** Beginner → Intermediate

## Learning Goals

By the end of this course, you will be able to:

- Explain what AI agents are and how they differ from chatbots and RAG systems
- Describe the Perceive-Decide-Act (PDA) loop and how it drives agent behavior
- Build a research agent using LangChain with web search and memory tools
- Design and implement a multi-agent content pipeline (Research → Writer → Editor)
- Apply production patterns: human-in-the-loop, rate limiting, sandboxing, and monitoring

## 📦 Deploy

### Step 1: Push to Your Own GitHub
1. [Create a new repository](https://github.com/new) on GitHub (name it `ai-agent-learning`)
2. Push the cloned repo to your account:
```bash
git remote set-url origin https://github.com/your-username/ai-agent-learning.git
git push -u origin main
```

### Step 2: Deploy

**GitHub Pages**
1. Go to your repo Settings → Pages
2. Source: Deploy from a branch → `main` → `/website`
3. Your site will be live at `https://your-username.github.io/ai-agent-learning/`

**Vercel**
1. Import repo at [vercel.com/new](https://vercel.com/new)
2. Root Directory: `website`
3. Framework: Other
4. Deploy!
