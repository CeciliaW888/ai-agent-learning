## 📦 部署

### 第一步：推送到你自己的 GitHub
1. 在 GitHub 上[创建一个新仓库](https://github.com/new)（命名为 `ai-agent-learning`）
2. 将克隆的仓库推送到你的账户：
```bash
git remote set-url origin https://github.com/YOUR_USERNAME/ai-agent-learning.git
git push -u origin main
```

### 第二步：部署

**GitHub Pages**
1. 进入你的仓库 Settings → Pages
2. Source：选择 Deploy from a branch → `main` → `/website`
3. 你的网站将在 `https://YOUR_USERNAME.github.io/ai-agent-learning/` 上线

**Vercel**
1. 在 [vercel.com/new](https://vercel.com/new) 导入仓库
2. Root Directory：`website`
3. Framework：Other
4. 点击部署！
