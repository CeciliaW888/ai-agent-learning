## 关于本课程

21天，21个小时，从AI用户到AI Agent开发者。

这是一套结构化的学习课程，带你从了解什么是AI Agent，到构建第一个LangChain Agent，再到部署一个多Agent内容创作流水线。每天1小时，以视频为主，动手实践，公开学习。

**时长：** 21天（3周）
**每日投入：** 1小时
**语言：** 英文
**难度：** 入门 → 进阶

## 学习目标

完成本课程后，你将能够：

- 解释什么是AI Agent，以及它与聊天机器人和RAG系统的区别
- 描述感知-决策-行动（PDA）循环如何驱动Agent行为
- 使用LangChain构建具备网络搜索和记忆功能的研究Agent
- 设计并实现多Agent内容流水线（研究 → 写作 → 编辑）
- 应用生产级模式：人机协作、速率限制、沙箱隔离和监控

```bash
# 打开课程网站
open website/index.html      # macOS/Linux
start website/index.html     # Windows
```

### 安装为PWA

1. 在 Chrome 或 Edge 中打开课程网站
2. 点击地址栏中的"安装"图标，或页面上的 **📲 安装应用** 按钮
3. 安装后，从桌面或主屏幕直接打开 — 支持离线访问

## 📦 部署

### 第一步：推送到你自己的 GitHub

1. 在 GitHub 上[创建一个新仓库](https://github.com/new)（命名为 `ai-agent-learning`）
2. 将克隆的仓库推送到你的账户：

```bash
git remote set-url origin https://github.com/your-username/ai-agent-learning.git
git push -u origin main
```

### 第二步：部署

**GitHub Pages**

1. 进入你的仓库 Settings → Pages
2. Source：选择 Deploy from a branch → `main` → `/website`
3. 你的网站将在 `https://your-username.github.io/ai-agent-learning/` 上线

**Vercel**

1. 在 [vercel.com/new](https://vercel.com/new) 导入仓库
2. Root Directory：`website`
3. Framework：Other
4. 点击部署！
