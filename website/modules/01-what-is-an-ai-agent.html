Day 1: What is an AI Agent?

🤖
The big shift: Most people use AI like a calculator — ask a question, get an answer. An AI Agent is different. It decides what to do next, uses tools to get things done, and keeps going until the job is finished.

📚 Core Concepts

🧠 What Makes Something an "Agent"?

An AI Agent is a system that can perceive its environment, decide what to do, and act — repeatedly, until a goal is achieved.

The key word is autonomous. A chatbot waits for you to ask questions. An agent keeps going on its own — searching the web, running code, calling APIs, writing files — whatever it takes to finish the job.

💡
Simple test: If you give it a goal and walk away, does it figure out the steps itself? If yes — it's an agent. If it needs you to guide every step — it's a chatbot.

🏗️ The LLM is Just the Brain

An LLM (like GPT-4 or Claude) by itself is just a very smart text predictor. It becomes an agent when you wrap it with:

🔧
Tools

Functions the agent can call — web search, code execution, database queries, sending emails.

🧠
Memory

Short-term (current conversation) and long-term (facts stored across sessions).

📋
Planning

The ability to break a big goal into steps and decide what to do next at each stage.

💬 Watching an Agent Think

Here's what it looks like when an agent processes a real request:

User: "Research the top 3 AI frameworks for building agents in 2024 and summarize the pros and cons."
Agent [Thought]: I need to search for current information. Let me use the web search tool.
Agent [Action]: search_web("best AI agent frameworks 2024")
Tool Result: [Returns results about LangChain, LlamaIndex, AutoGen...]
Agent [Thought]: Good results. Now I'll search for comparison articles to get the pros/cons.
Agent [Action]: search_web("LangChain vs LlamaIndex vs AutoGen comparison")
Agent [Thought]: I have enough information. Now I'll synthesize this into a clear summary.
Agent [Answer]: Here are the top 3 AI agent frameworks in 2024...

Notice the loop: Think → Act → Observe → Think again. This is the core of every agent system you'll build in this course.

🤖 Agent vs Chatbot — The Key Difference

💬 Chatbot

Responds to one message at a time. Waits for you to tell it what to do next. Can't use external tools or take actions in the real world.

"What's the weather today?" → gives an answer (from training data, often outdated)

⚠️
Common misconception: "Isn't ChatGPT an agent?" — Not exactly. ChatGPT (without plugins) is a very capable chatbot. ChatGPT with web browsing + code interpreter + file access starts to become an agent. The tools are what make the difference.

📖 Key Terms

AI Agent
An autonomous AI system that perceives, decides, and acts repeatedly to achieve a goal — without needing a human to guide every step.
LLM (Large Language Model)
The "brain" of an agent — e.g. GPT-4, Claude, Gemini. Handles reasoning and language. Needs tools + memory to become an agent.
Tool / Function Call
A capability the agent can use: search the web, run Python, query a database, send an email. Tools are what give agents real-world reach.
Autonomy
The ability to take initiative — deciding what steps to take without being told explicitly. The defining trait of an agent vs a chatbot.
ReAct Pattern
Reason + Act — the loop where the agent thinks about what to do, does it, observes the result, and thinks again. You'll build this in Week 2.
